# Boost Agency Website

This is a modern, responsive website template for Boost Agency, a web development subscription service. The design is inspired by DesignJoy but customized for web development services.

## Features

- Clean, modern design
- Fully responsive layout
- Subscription-based service presentation
- Pricing section
- FAQ section
- Call-to-action elements
- Testimonials section
- Service overview

## Getting Started

1. Clone or download this repository to your local machine
2. Open `index.html` in your web browser to view the website
3. Customize the content and styling to fit your needs

## Customization

### Changing Colors

You can easily change the color scheme by editing the CSS variables in the `:root` selector in `styles.css`:

```css
:root {
    --primary-color: #0066FF; /* Main brand color */
    --secondary-color: #111111; /* Secondary color */
    --text-color: #333333; /* Main text color */
    --light-text: #777777; /* Light text color */
    --background-color: #FFFFFF; /* Main background color */
    --light-bg: #F5F5F7; /* Light background color */
    --border-color: #EEEEEE; /* Border color */
}
```

### Updating Content

1. Edit the `index.html` file to update text content
2. Replace services, features and descriptions to match your offering
3. Update pricing and FAQ sections to reflect your business model

### Images

To add images to the website:

1. Create an `images` folder in the root directory
2. Add your images to this folder
3. Reference them in your HTML using relative paths, e.g., `<img src="images/your-image.jpg" alt="Description">`

### Adding Pages

To add more pages:

1. Create a new HTML file in the root directory (e.g., `about.html`, `contact.html`)
2. Copy the basic structure from `index.html`
3. Update the navigation links in all pages to include links to your new pages

## Deployment

To deploy this website:

1. Upload all files to your web hosting service
2. Make sure to maintain the same directory structure
3. Set `index.html` as the default page
